# /panels

## Name
/panels

## Description
This method returns a list of panels that are associated with the user.

## Parameters
This method does not take any parameters.

## Authentication
`User-Token` is required.

## Example Request
`GET /panels`

## Example Response
```json
[
  {
    "panel_serial": "AAAAAA",
    "alias": "Home",
    "type": "PMax"
  }
]
```
