# /customer_info

## Name
/customer_info

## Description
This method returns customer information for the specified panel.

## Parameters
This method does not take any parameters.

## Authentication
`User-Token` is required.

## Example Request
`GET /customer_info`

## Example Response
```json
{
  "account": "000000",
  "name": "Joe Bloggs",
  "address": null,
  "phone": null,
  "email": null,
  "remarks": null
}
```
