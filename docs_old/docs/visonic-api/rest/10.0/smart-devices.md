# /smart_devices

## Name
/smart_devices

## Description
This method returns basic smart device information.

## Parameters
This method does not take any parameters.

## Authentication
`User-Token` is required.

`Session-Token` is required.

## Example Request
`GET /smart_devices`

## Example Response
```json
[
  {
    "partition": 1,
    "indoor_temperature": 11.5
  }
]
```
