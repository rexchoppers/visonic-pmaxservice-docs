# PmaxService/getDate

## Name
PmaxService/getDate

## Description
This method returns the panel's current date

## Parameters
This method does not take any parameters.

## Example Request
```json
{
    "params": [],
    "jsonrpc": "2.0",
    "method": "PmaxService/getDate",
    "id": 1
}
```

## Example Response
```json
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": "20231107"
}
```
