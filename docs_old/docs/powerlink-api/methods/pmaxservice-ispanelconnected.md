# PmaxService/isPanelConnected

## Name
PmaxService/isPanelConnected

## Description
This method returns the panel's current connection status

## Parameters
This method does not take any parameters.

## Example Request
```json
{
    "params": [],
    "jsonrpc": "2.0",
    "method": "PmaxService/isPanelConnected",
    "id": 1
}
```

## Example Response
```json
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": true
}
```
