# PmaxService/getApiVersion

## Name
PmaxService/getApiVersion

## Description
This method returns the API version of the module.

## Parameters
This method does not take any parameters.

## Example Request
```json
{
    "params": [],
    "jsonrpc": "2.0",
    "method": "PmaxService/getApiVersion",
    "id": 1
}
```

## Example Response
```json
{
    "id": 1,
    "jsonrpc": "2.0",
    "result": "3.31"
}
```
