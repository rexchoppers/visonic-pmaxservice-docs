# PmaxService/getLocationsList

## Name
PmaxService/getLocationsList

## Description
This method returns a full list of locations on the panel

## Parameters
This method does not take any parameters.

## Example Request
```json
{
    "params": [],
    "jsonrpc": "2.0",
    "method": "PmaxService/getLocationsList",
    "id": 1
}
```
