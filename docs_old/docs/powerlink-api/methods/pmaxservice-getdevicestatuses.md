# PmaxService/getDeviceStatuses

## Name
PmaxService/getDeviceStatuses

## Description
This method returns all status information for a device.

## Parameters
- `params[0]` PowerG Device type ID
- `params[1]` Device ID

## Example Request
```json
{
    "params": [
        3, 3
    ],
    "jsonrpc": "2.0",
    "method": "PmaxService/getDeviceStatuses",
    "id": 1
}
```
